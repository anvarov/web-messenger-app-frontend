# Installation

```bash
git clone https://github.com/anvarov/web-messenger-app-frontend.git
cd web-messenger-app
```

2. Install dependencies

Optional

```bash
yarn cache clean
yarn install
```

3. Build Web Messenger

```bash
yarn run build
```

# Usage

In root folder:

```bash
yarn serve
```

This will start the server and application will be available at
http://localhost:3000
